post_seq

Receive a Pulseq .seq file in the browser (local processing only).

Drop a .seq file here or browse

Only the file name and contents are read in your browser; nothing is uploaded to a server.

Expected API for scripts: POST multipart field name file (see README). GitHub Pages does not run a server; use these examples with a backend that implements the same contract, or rely on this page for manual checks.
