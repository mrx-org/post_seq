post_seq

Receive a Pulseq .seq file in the browser (local processing only).

Drop a .seq file here or browse

Only the file name and contents are read in your browser; nothing is uploaded to a server.

Open with a link: ?url= (HTTPS URL to a .seq; page fetches it if CORS allows) or #b64= (base64url of file bytes; size-limited). Tiny snippets: ?seq= (short text only). POST field file for APIs (see README).
